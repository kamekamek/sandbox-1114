# sandbox-1114

焼き鳥屋ランディングページ - 2バージョン展開プロジェクト

## 🍢 プロジェクト概要

このリポジトリでは、焼き鳥屋の2つの異なるコンセプトのランディングページを管理しています。

- **RPG版** (`rpg/`): ファンタジーRPG風の世界観で焼き鳥を表現
- **通常版** (`shop/`): モダンでスタイリッシュな焼き鳥ショップ

## 📂 ディレクトリ構成

```
sandbox-1114/
├── rpg/           # RPG風ランディングページ
│   └── index.html
├── shop/          # 通常版ランディングページ
│   ├── index.html
│   ├── script.js
│   └── styles.css
└── SETUP_INSTRUCTIONS.md  # セットアップ詳細指示書
```

## 🌐 公開URL（GitHub Pages設定後）

- **RPG版**: https://kamekamek.github.io/sandbox-1114/rpg/
- **通常版**: https://kamekamek.github.io/sandbox-1114/shop/

## 🚀 セットアップ方法

詳細な手順は [SETUP_INSTRUCTIONS.md](./SETUP_INSTRUCTIONS.md) を参照してください。

### クイックスタート

1. 各ブランチで適切なディレクトリにファイルを配置
2. mainブランチに両方をマージ
3. GitHub PagesでmainブランチのrootをSource指定

## 🔧 開発

各バージョンは独立したディレクトリで管理されているため、競合なく並行開発が可能です。
